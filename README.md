# class-assignments

This repository contains three programming assignments created using p5.js. Each assignment explores different interactive and visual concepts.

## Assignment 1 (Lab 6)

This assignment demonstrates the use of arrays to control multiple moving objects on the screen. Rectangles move horizontally across the canvas at different speeds, and when they reach the edge, they loop back to the start. One of the shapes is also controlled by the mouse, allowing for user interaction.

## Assignment 2 (Lab 8)

This assignment focuses on working with images and user interaction. Multiple images are loaded and displayed on the canvas, including a transparent PNG. The user can move the mouse to interact with the images, creating a dynamic visual experience.

## Assignment 3 (Assignment 5)

This assignment creates an interactive TV interface using shapes and user input. Users can click buttons to turn the TV on/off, adjust the volume, and activate a hidden “secret mode” using both mouse and keyboard input. It demonstrates conditionals, interactivity, and UI design.

## Notes

All assignments were originally created for class and uploaded as ZIP files for submission. They showcase different programming concepts including arrays, image handling, and interactive controls.
